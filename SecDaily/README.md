## 知识早餐脚本

**就像身体需要吃早餐，头脑也需要补充知识**

每个早晨，打开脚本，让它帮你获取先知、安全客、freebuf、嘶吼的主站咨询。

### 特性

* 爬取框架基本完善，任务结构化
* 添加新目标时以xpath表达式作为特征库
* 输出形式为html，方便点击访问
* 下一步将引入协程机制，加快爬取效率
    
### 依赖库

暂时用到的第三方模块有requests、lxml、jinja2
